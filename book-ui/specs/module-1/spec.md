# Module 1: The Robotic Nervous System (ROS 2)

## Overview
This module covers Physical AI fundamentals and ROS 2 middleware for robot control. Students will learn the transition from digital AI to embodied intelligence and master ROS 2 communication patterns.

## Learning Objectives
- Understand Physical AI and embodied intelligence concepts
- Master ROS 2 architecture (Nodes, Topics, Services, Actions)
- Develop ROS 2 packages using Python with rclpy
- Bridge AI agents to ROS controllers
- Create and understand URDF for humanoid robots

## Topics

### Chapter 1: Introduction to Physical AI
- Digital AI vs Physical AI
- Embodied Intelligence concepts
- Perception, Control, and Learning systems
- The humanoid advantage
- Sim-to-Real challenges

### Chapter 2: ROS 2 Fundamentals
- ROS 2 architecture and middleware
- Nodes, Topics, Services, Actions
- Quality of Service (QoS) profiles
- Launch files and parameters
- URDF (Unified Robot Description Format)
- AI-ROS integration with rclpy

## Prerequisites
- Python programming (intermediate)
- Basic Linux command line
- Understanding of AI/ML concepts

## Duration
Estimated: 8-10 hours

## Assessment
- Practice exercises for each chapter
- Hands-on ROS 2 package development
- URDF model creation
- AI bridge node implementation
