# Module 2: The Digital Twin (Gazebo & Unity) + NVIDIA Isaac

## Overview
This module covers simulation environments for Physical AI, including Gazebo for physics simulation, Unity for high-fidelity rendering, and NVIDIA Isaac for AI-powered robotics.

## Learning Objectives
- Understand digital twin concepts in robotics
- Configure physics simulations in Gazebo
- Simulate sensors: LiDAR, depth cameras, IMUs
- Use NVIDIA Isaac Sim for photorealistic simulation
- Implement Isaac ROS for GPU-accelerated perception
- Configure Nav2 for robot navigation

## Topics

### Chapter 3: Digital Twin (Gazebo & Unity)
- Digital Twin paradigm in robotics
- Gazebo physics simulation (gravity, collisions, materials)
- SDF (Simulation Description Format)
- Unity for high-fidelity visualization
- Sensor simulation (LiDAR, cameras, IMUs)
- Sim-to-real transfer challenges

### Chapter 4: NVIDIA Isaac
- Isaac ecosystem overview
- Isaac Sim on Omniverse platform
- USD scene representation
- RTX ray tracing for photorealism
- Synthetic data generation
- Isaac ROS packages
- Visual SLAM (VSLAM)
- Nav2 path planning

## Prerequisites
- Completed Module 1
- Ubuntu 22.04 with NVIDIA GPU
- Basic understanding of 3D graphics

## Duration
Estimated: 12-15 hours

## Assessment
- Gazebo world creation
- Sensor simulation setup
- Isaac Sim environment design
- Navigation stack configuration
